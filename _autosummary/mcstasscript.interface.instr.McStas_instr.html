mcstasscript.interface.instr.McStas_instr¶

class mcstasscript.interface.instr.McStas_instr(name, **kwargs)¶

Main class for writing a McStas instrument using McStasScript

Initialization of McStas_instr sets the name of the instrument file and its methods are used to add all aspects of the instrument file. The class also holds methods for writing the finished instrument file to disk and to run the simulation.
